# Mon Pokedex

Une application simple pour afficher les informations des Pokémon.

## Fonctionnalités
- Recherche de pokémon 
- Système de favoris

## Installation
```bash
# Cloner le projet
git clone 

# Installer les dépendances
npm install

# Lancer l'application
npm start
```

## Technologies
- React
- PokeAPI
- CSS/SCSS
