# ReactPokedexArchi

Application de type Pokédex développée avec React, illustrant une architecture propre, des composants modulaires et des bonnes pratiques vues en cours.

## Description

Cette application permet de :  
- Consulter la liste de tous les pokémons
- Saisir le nom d'un pokémon
- Passage en version chromatique 
- Afficher les détails d’un Pokémon  
- Naviguer entre liste et fiche Pokémon  
- Ajouter et gérer ses Pokémon favoris  

Objectifs pédagogiques : architecture React, appels API, routing, gestion d’état, optimisation des performances, amélioration progressive.

## Installation & Lancement

```bash
# Cloner le dépôt
git clone https://github.com/evboys/ReactPokedexArchi.git
cd ReactPokedexArchi

# Installer les dépendances
npm install
# ou yarn install

# Lancer en mode développement
npm start
# Accès à http://localhost:3000

# Construire la version production
npm run build
```
